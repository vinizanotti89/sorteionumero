🎲 Projeto Sorteio de Números

O objetivo inicial era simples: construir um site que sorteasse números de forma aleatória.
Mas como sempre... não resisti em deixar o projeto com uma pitada de personalidade!

🚀 Projeto inicial simples, mas com 100% de autoria e criatividade aplicada. Cada linha foi escrita pensando em ir além do básico.

📄 Descrição do Projeto

- Geração de números aleatórios dentro de um intervalo definido pelo usuário.

- Interface amigável e direta.

- Cada detalhe visual, inclusive imagens e ícones, foi feito com escolha própria (nada copiado!).

Resumo:
Um projeto focado em lógica básica de JavaScript, interação simples com usuário e experimentação visual personalizada.

🎯 Funcionalidades Implementadas

- Definição de intervalo de sorteio (número mínimo e máximo).

- Geração instantânea de números aleatórios.

- Estilização visual para deixar o projeto único.

- Responsividade básica para funcionar em diferentes tamanhos de tela.

🔥 Extras (história pessoal no projeto)
Esse projeto me fez entender como coisas simples escondem lógicas mais profundas.
No começo, parecia "só sortear um número", mas fui entendendo como lidar com entradas erradas, como deixar o usuário guiado, e principalmente: como fazer o JavaScript trabalhar pra mim, não contra mim 😅
